1. Description La fonction infobulle est une fonction javascript permettant d'afficher très simplement des informations à côté du curseur lors du survol d'un élément (liens, image, div,...).
2. Fichiers nécessaires infobulle.js (télécharger) : contient la fonction javascript infobulle
infobulle.css (télécharger) : contient la déclaration CSS du style appliqué à l'infobulle
3. Configuration Aucune configuration nécessaire.
Vous pouvez modifier l'apparence des infobulles et de leur contenu dans le fichier css.
4. Utilisation La fonction javascript doit être appelée ainsi sur un élément (ici un lien) : <a onMouseOver="infobulle(this, 'texte à afficher');">Test</a>
