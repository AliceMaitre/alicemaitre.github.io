<fieldset><legend>1. Description</legend>
La fonction infobulle est une fonction javascript permettant d'afficher très simplement des informations à côté du curseur lors du survol d'un élément (liens, image, div,...).
</fieldset>

<fieldset><legend>2. Fichiers nécessaires</legend>
<strong>infobulle.js</strong> (<a href="?file=infobulle.js">télécharger</a>) : contient la fonction javascript infobulle<br />
<strong>infobulle.css</strong> (<a href="?file=infobulle.css">télécharger</a>) : contient la déclaration CSS du style appliqué à l'infobulle<br />
</fieldset>

<fieldset><legend>3. Configuration</legend>
Aucune configuration nécessaire.<br />
Vous pouvez modifier l'apparence des infobulles et de leur contenu dans le fichier css.
</fieldset>

<fieldset><legend>4. Utilisation</legend>
La fonction javascript doit être appelée ainsi sur un élément (ici un lien) :
<xmp><a onMouseOver="infobulle(this, 'texte à afficher');">Test</a></xmp>
</fieldset>
